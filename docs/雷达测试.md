
适用对象：Orin 算力机（192.168.71.51）上，通过 Standard Robots AMR ROS2 SDK v1.3.0 读取 Helios 底盘前后 2D 激光雷达数据。  
验证环境：ROS2 Humble、底盘 IP 192.168.71.50、地图 AB_0619、定位正常（location_state=3）。  
文档日期：2026-06-25


---

1. 概述

珞石 Helios 轮臂机器人的移动底盘配备 前后各一台 2D 激光雷达，由底盘 SROS 系统用于激光定位与避障。Orin 侧通过 完整版 `sr_amr_control` 节点（SDK v1.3.0）可将雷达数据以 ROS2 标准消息 `sensor_msgs/LaserScan` 发布。

9. 重要限制说明

1. 非原始扫描：SDK 从 SROS 的 `loc_laser_points`（处理后障碍点）重建 `LaserScan`，不是驱动层原始 ring data。
2. 无 PointCloud2：当前 ROS2 节点只发布 `LaserScan`，不发布 `sensor_msgs/PointCloud2`。
3. 默认关闭：雷达上报默认 `false`，重启节点后需重新开启（或在 launch 中加 `lidar:=true`）。
4. TF 依赖：使用默认 `right_front_laser_link` / `left_behind_laser_link` 时必须提供对应 TF。
5. 与导航独立：读取雷达数据不影响 MATRIX 定位和导航功能，但雷达故障会导致无法启动移动任务。
